01 — About

Background

I'm a computational researcher specializing in computer-aided drug design (CADD), with a focus on structure-based approaches including molecular docking, molecular dynamics simulations, and structural bioinformatics.

My work bridges computational chemistry and medicinal chemistry — building pipelines that translate structural data into actionable hypotheses for drug development. I'm particularly interested in virtual screening, protein–ligand interaction analysis, and leveraging PDB crystal structure data to understand binding geometries.

I work primarily in Linux environments with Python-based toolchains, integrating open-source cheminformatics libraries with domain-specific simulation software.

04 — Projects

Selected Work

P — 01
Virtual Screening Pipeline
SMILES-based compound filtering pipeline integrating PAINS, BRENK, NIH, and ZINC rules via RDKit FilterCatalog, followed by automated docking and post-docking rescoring.
Python RDKit AutoDock Vina Bash
P — 02
Interaction Geometry Knowledge Base
Data-driven tool that extracts bonded interaction geometry statistics from PDB crystal structures, building a custom knowledge base analogous to CSD Mogul for validating docking poses.
Python Biopython MDAnalysis PDB
P — 03
LC3B MD Trajectory Analysis
Batch GROMACS trajectory analysis computing partial RMSD of a peptide ligand near PHE52 across multiple simulation runs, with automated reporting and visualization.
GROMACS MDAnalysis Python Matplotlib
P — 04
AhR Structure-Based Docking
Structure-based virtual screening campaign targeting the Aryl Hydrocarbon Receptor (AhR), including protein preparation, binding site definition, and PLIP-based interaction profiling.
AutoDock Vina PLIP PyMOL RDKit
